# Joseph Danza Platform

Plataforma de video estilo "Netflix" para la academia Joseph Danza.

Stack:
- Next.js 14 + TypeScript + App Router
- Tailwind CSS
- Prisma + PostgreSQL
- NextAuth (Email + Google)
- Mux (VOD, HLS, player)
- Stripe (suscripciones)
- Hosting recomendado: Vercel
